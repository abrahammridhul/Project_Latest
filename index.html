About FloodSafe

FloodSafe is a lightweight client-side flood warning dashboard intended for quick reporting, basic weather checks and emergency contact access. It stores alerts locally (in your browser). For live weather data you can add an OpenWeatherMap API key on the Weather page.

Contact / Helplines

  • Fire & Rescue: 101
  • Police: 100
